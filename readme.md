# Laravel Project

> Laravel 5.6

## Quick Start

``` bash
# Install Dependencies
composer install

# Install Node Package Manager
npm install

# Run Migrations
php artisan migrate

# Import database data
php artisan db:seed

# Add virtual host if using Apache

# If you get an error about an encryption key
php artisan key:generate
```
